# RL002 — main-screen-scale

Default severity: **warning**. Automatic fix: **limited**.

## What it finds

RL002 reports `UIScreen.main.scale` and `UIScreen.main.nativeScale`.

```swift
let pixelWidth = width * UIScreen.main.scale
```

## Adaptive alternative

Prefer the trait environment that owns the rendered content:

```swift
final class CanvasView: UIView {
    func render(width: CGFloat) {
        let pixelWidth = width * traitCollection.displayScale
    }
}
```

A scene-local screen passed from the caller is also valid when the operation genuinely needs screen information.

## Safe fix boundary

`resizelint fix` replaces an exact `UIScreen.main.scale` expression only inside an instance member of a proven UIKit view or view-controller type where `traitCollection` is directly available. It never rewrites `nativeScale`, static members, free functions, or ambiguous types.
