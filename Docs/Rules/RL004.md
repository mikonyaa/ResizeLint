# RL004 — global-window-access

Default severity: **error**. Automatic fix: **none**.

## What it finds

RL004 reports process-global attempts to select an arbitrary current window or scene.

```swift
let window = UIApplication.shared.windows.first
let scene = UIApplication.shared.connectedScenes.first
```

It also recognizes common compact-map chains that select the first `UIWindowScene` and its first window.

## Adaptive alternative

Carry ownership from the caller:

```swift
func presentInspector(from viewController: UIViewController) {
    guard let windowScene = viewController.view.window?.windowScene else { return }
    // Use the scene that owns the presenting view controller.
}
```

Deliberate iteration over every scene or every window for a broadcast operation is not reported when no arbitrary element is selected as current UI context.
