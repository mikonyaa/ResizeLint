# RL007 — orientation-layout-decision

Default severity: **warning**. Automatic fix: **none**.

## What it finds

RL007 reports layout branches driven by interface or device orientation, including `isPortrait` and `isLandscape` checks.

```swift
let isWide = view.window?.windowScene?.interfaceOrientation.isLandscape == true
columns = isWide ? 4 : 2
```

## Adaptive alternative

Base layout on the container:

```swift
let isWide = view.bounds.width >= 720
columns = isWide ? 4 : 2
```

Supported-orientation declarations, camera or motion coordinate conversion, parameter declarations, and values passed directly into a system API without calculating layout are excluded.
