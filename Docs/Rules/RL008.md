# RL008 — legacy-app-lifecycle

Default severity: **error**, only when the legacy condition is proven. Automatic fix: **none**.

## What it finds

RL008 is a project-level diagnostic. It requires all of the following evidence:

- an iOS application target;
- `UIApplicationDelegate.window`;
- no SwiftUI `@main App`;
- no `UISceneDelegate`;
- no scene manifest in the relevant property list;
- no generated scene-manifest build setting.

```swift
final class AppDelegate: UIResponder, UIApplicationDelegate {
    var window: UIWindow?
}
```

## Adaptive direction

Adopt scene lifecycle and place window-specific UI state in the scene delegate or SwiftUI scene hierarchy. This is an architectural migration, so ResizeLint provides evidence and guidance but never performs an automatic rewrite.

If target mapping is incomplete or ambiguous, ResizeLint records an audit notice in verbose output instead of emitting an error.
