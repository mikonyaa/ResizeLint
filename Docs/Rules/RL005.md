# RL005 — global-status-bar-geometry

Default severity: **error**. Automatic fix: **none**.

## What it finds

RL005 reports legacy process-global status-bar geometry and orientation:

```swift
let height = UIApplication.shared.statusBarFrame.height
let orientation = UIApplication.shared.statusBarOrientation
```

## Adaptive alternative

Prefer safe areas for layout. When status-bar state is genuinely needed, obtain it from the owning scene:

```swift
let topInset = view.safeAreaInsets.top
let frame = view.window?.windowScene?.statusBarManager?.statusBarFrame
```

A scene-local `statusBarManager` is not an RL005 error. Orientation-driven layout may still be reported separately by RL007.
